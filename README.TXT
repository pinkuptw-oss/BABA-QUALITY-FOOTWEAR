BABA QUALITY FOOTWEAR - Showcase Website

Files included:
- index.html
- styles.css
- script.js
- README.txt

What this site does:
- A responsive, static showcase website for your shop "BABA QUALITY FOOTWEAR".
- Categories: Mens, Ladies, Kids, Accessories.
- "Shop Now" and "View Products" buttons are placeholders until you provide your Vyapar store link.

How to add your Vyapar store link:
1. Open index.html in a text editor.
2. Find the button you want to update (for example the "Visit Vyapar Store" button).
3. Replace the button's href="#" and onclick attribute with your Vyapar link, for example:
   <a class="btn btn-primary" href="https://your-vyapar-store-link.example">Visit Vyapar Store</a>
4. Save the file.

How to host:
- Upload the files to any static hosting (GitHub Pages, Netlify, Vercel, or a regular web host).
- Or open index.html directly in a browser to preview locally.
